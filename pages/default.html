This HTML template will help you get started quickly. It has a few features which you'll find useful:

  • A sidebar flyout (exposed by the wrench icon)
  • A page selector across the top, with pages coded in HTML and JS
  • jQuery, jQuery UI, Font-Awesome and Select2 all included
  • The Domo spinner icon is included - just call showSpinner() and hideSpinner()
  • Basic fading transitions between pages
  • Headers animate in and out as they change

Simply fork this repository in bitbucket, clone it down, and ensure you've met all the other dependencies as detailed in the Domo App SDK documentation.
